## 包含的工具包
- 添加以下命令行工具

命令行 | 备注
---|---
curl | 基础命令
wget | 基础命令
git | 用于获取仓库信息
helm | 用户打包Chart包
maven | 用于测试及打包Java程序
img | 用于镜像构建阶段
docker | 用于镜像构建阶段
kaniko | 用于镜像构建阶段
mysql-client | 用于数据库初始化脚本测试
xmlstarlet | 用户HAP框架配置文件修改、POM文件修改
sonar-scanner | 用于代码扫描
yq | 用于yaml文件解析
jq | 用于json文件解析
